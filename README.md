# bright-naver-dictionary-chrome-extension

## Description

네이버 사전은 브라우저 테마를 무조건 따라가게 돼있어서 다크 테마이면서 네이버 사전만 밝게 쓰는게 불가능했는데, 이 크롬 익스텐션을 사용하면 네이버 사전을 항상 밝게 사용할 수 있습니다.
